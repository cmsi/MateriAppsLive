MateriApps LIVE! LTX Edition

[English/日本語]

The latest version of this document is available at https://github.com/cmsi/MateriAppsLive/wiki/MateriAppsLive-ltx-en.

MateriApps LIVE! LTX Edition is a virtual disk image for VirtualBox, in which the Japanese LaTeX environment is included in addition to the standard VirtualBox edition of MateriApps LIVE!. It is being developed and distributed as a hands-on environment of materials science application programs and also as a training environment of UNIX (UNIX commands, network, Emacs, Gnuplot), LaTeX, programming language (C, C++, Fortran), etc.

  • What's MateriApps?

    A number of high-performance and high-precision application software for computational materials science simulation have been developed in Japan. The experimentalists and researchers in the private companies, however, don't even know the existence of such applications, and are mainly using open-source software from abroad or commercial software. MateriApps, a portal site of materials science simulation, is introducing the features and characteristics of various materials science simulation programs that have not been known widely or that are being developed in years to come.

  • What's MateriApps LIVE!?

    In the meantime, when trying to try out open-source software, installation of software itself often becomes a serious obstacle. MateriApps LIVE! offers an environment where one can try out computational materials science simulation freely, using a notebook PC, etc. All environment required to begin tutorials, such as MateriApps applications, OS (Debian GNU/Linux), editors, and visualization tools, is provided in a USB memory stick.

  • What's VirtualBox?

    VirtualBox is open-source software providing a virtual machine environment. VirtualBox can be installed on Linux、Mac OS X、Windows、Solaris. It supports most popular operating systems on x86/x64 architecture, such as FreeBSD、Linux、OpenBSD、OS/2 Warp、Windows、Mac OS X、Solaris, etc, as guest OS. VirtualBox is distributed under GNU General Public License (GPL) Version2.

Contents of USB stick memory

  • VirtualBox-*-OSX.dmg : VirtualBox Installer for Mac OS X (version 5)
  • VirtualBox-*-Win.exe : VirtualBox Installer for Windows (version 5)
  • MateriAppsLive-ltx-*-amd64.ova : MateriApps LIVE! virtual disk image (64bit version)
  • README-en.html : This file
  • setup-en.pdf : How to setup MateriApps LIVE!
  • vbconfig.bat, vbconfig.command : Scripts for disabling noisy popup menus in VirtualBox
  • MD5SUM : Checksum

How to setup MateriApps LIVE!

  1. Copy contents of USB memory stick to hard drive

    Copy all the files to your PC (e.g., Desktop).

  2. Starts VirtualBox Installer and install VirtualBox software.

  3. Starts VirtualBox by double clicking MateriAppsLive-ltx-*-amd64.ova Virtual Disk Image. Click "Import" button in VirtualBox import window.
  4. Disable noisy popup menus in VirtualBox
    • Windows: double click vbconfig.bat
    • maxOS: double click vbconfig.command or execute the following in the Terminal. sh vbconfig.command
    • To enable popups again, choose "Reset All Warnings" from "Help" menu in VirtualBox.

  5. Setup file transfer between host OS and MateriApps LIVE!

    Please refer to "How to transfer files from/to MateriApps LIVE!" below.

  6. Choose "MateriAppsLive" on Oracle VM VirtualBox Manager panel and click "Start" button.

  7. Login to MateriApps LIVE! by using the following username and password
    • Login(user name): user
    • Password: live

How to transfer files from/to MateriApps LIVE!

  1. Transfer files between Host OS (macOS X or Windows) and MateriApps LIVE!

    1. Choose "MateriAppsLive" on Oracle VM VirtualBox Manager panel and click "Settings"
    2. Open "Shared Folders" tab, and click "+" icon on the right to add a new shared folder definition
    3. Click "v" sign on the right of "Folder Path:" field, and choose "Other". In the file dialog box, select a folder being shared between Host OS and MateriApps LIVE!
    4. Enable "Auto-mount" check box and press "OK" button.
    5. After booting (rebooting) the virtual machine, you can see the folder selected in step 3 under /media/sf_... in MateriApps LIVE!

  2. Connect a USB stick memory to MateriApps LIVE!

    • VirtualBox USB settings: https://www.virtualbox.org/manual/ch03.html#settings-usb
